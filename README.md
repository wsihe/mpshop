# Mpshop - 微信小程序端

> 基于 Mpvue 的小程序

### 项目运行

```
git clone https://github.com/wsihe/mpshop.git

cd mpshop

npm install

npm run dev

```

### 功能

- 首页商品列表
- 商品详情
- 购物车
- 会员中心
